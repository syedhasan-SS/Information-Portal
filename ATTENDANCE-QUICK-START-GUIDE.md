### "Location Permission Denied"
**Problem**: Can't login/logout because location is blocked

**Solution**:
1. Click the location icon in your browser's address bar
2. Change "Location" to "Allow"
3. Refresh the page
4. Try login/logout again

**Browser-Specific**:
- **Chrome**: Click ğŸ”’ â†’ Site Settings â†’ Location â†’ Allow
- **Firefox**: Click ğŸ”’ â†’ Permissions â†’ Location â†’ Allow
- **Safari**: Safari â†’ Preferences â†’ Websites â†’ Location â†’ Allow

### "Already Logged In"
**Problem**: Can't login because there's already an active session

**Solution**:
1. You may have forgotten to logout last time
2. Click the "Logout" button first
3. Then click "Login" to start a new session

**If still stuck**: Contact your manager to mark the old session as incomplete

### Timer Not Updating
**Problem**: The work duration timer isn't counting

**Solution**:
1. Refresh your browser page
2. The session should still be active
3. Timer will resume from the correct time

### Can't See Logout Button
**Problem**: Only see "Login" button but you're already logged in

**Solution**:
1. Refresh the page
2. Check the Attendance history page to verify your status
3. If you have an active session, the logout button should appear

### Missing Logout from Yesterday
**Problem**: Forgot to logout yesterday

**Solution**:
1. Contact your manager
2. They can mark the session as incomplete
3. They can add a note about the actual logout time
4. No need to worry - it happens!
